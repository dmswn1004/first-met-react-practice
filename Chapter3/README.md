### 3.1 JSX란?


**JSX** : 자바스크립트의 확장 문법  
  - ex) 자바스크립트 코드와 HTML 코드가 결합되어 있는 JSX 코드
~~~javaScript
const element = <h1>Hello, world!</h1>;
~~~
---

### 3.2 JSX의 역할  
- 내부적으로 XML/HTML 코드를 자바스크립트로 변환하는 과정 거침
- 리액트의 **createElement()** 라는 함수가 JSX 코드를 자바스크립트 코드로 변환하는 역할을 함
